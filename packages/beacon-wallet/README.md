# `@tezos-x/beacon-wallet`

This package is part of the `@tezos-x/beacon-sdk` project. [Read more](https://github.com/airgap-it/beacon-sdk)

## Introduction

Use this package in your wallet to instanciate a `WalletClient` object and communicate to dApps.

## Usage

```
import { WalletClient } from '@tezos-x/beacon-wallet'

const walletClient = new WalletClient({ name: "Example Wallet" });
```

Check our documentation for more information. [Documentation](https://docs.walletbeacon.io)
